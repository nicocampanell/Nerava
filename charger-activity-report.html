Data Methodology

Charger availability data is collected every 15 minutes via TomTom's EV Charging Availability API across 25 stations in three Austin-metro zones. Data points include total ports, available ports, occupied ports, and out-of-service ports per station. Collection began March 30, 2026 and runs continuously. Session estimates are derived from occupied port snapshots and average session duration. This report contains 16,784 individual data points.
