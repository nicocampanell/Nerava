# @nerava/sdk

Official TypeScript SDK for the Nerava EV charging intelligence and incentive platform.

> **Scaffold only — Step 1 of a 20-step build plan.**
> Steps 1–11 build this SDK package incrementally.
> Steps 12–20 build the sibling `nerava-appstore` React package (see `../nerava-appstore`).
