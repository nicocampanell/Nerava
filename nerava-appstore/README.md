# nerava-appstore

In-app store experience that renders inside the EVject driver app. EVject is the shell, Nerava powers what is inside.

> Step 1 scaffold only. The Vite + React + TypeScript + Tailwind project is created in Step 12.
> This directory is intentionally near-empty until then — only this README and a placeholder
> `.gitkeep` live here right now so the directory is tracked in git.

## Planned stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- No localStorage anywhere (React state only)
- 390px viewport, phone-frame border

## Planned screens

1. Home (featured banner, category tabs, app grid)
2. App Detail (permissions, enable toggle)
3. My Apps (enabled apps only)
4. Settings (placeholder)

Full details of the app catalog, color system, and screen specs live in the project prompt.
